# Contributing

Contributions are welcome. For this project, keep changes simple and focused.

## Development guidelines

- Use C++20-compatible code.
- Keep console prompts clear and consistent.
- Do not commit generated runtime CSV files containing real account/contact data.
- Build before opening a pull request.

## Suggested checks

```bash
make
```

or

```bash
cmake -S . -B build
cmake --build build
```
